# Turing PCB

This is a PCB for a [Turing Machine](https://en.wikipedia.org/wiki/Turing_machine) that is made primarily using "simple" components, without using complex microchips or programmable stuff (as that would be cheating). It is designed to be programmed through the transition table being bridged by diodes (e.g. LEDs) in order to represent what is being done and how the machine should progress. It has both an automatic clock and the ability to step through it, to enable debugging programs or just run through them. It has 4 state bits and 8 tape bits. It ended up being much more complex than I thought at first, and took way longer than expected, but it was really enjoyable. It has 6 layers & 2 flood fills for both GND and VCC.

I made this project primarily for the fun of it. I've always found challenges with confounding variables fun and interesting, and believe that this is just sort of the next step in that regard, with Turing Machine's being fun proofs of concept and this being a very restricted form of it.
